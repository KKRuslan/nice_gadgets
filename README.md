# Nice Gadgets

A comprehensive E-commerce web application for discovering, filtering, and purchasing tech devices. Built with React, Redux Toolkit, and TypeScript, this project demonstrates a scalable front-end architecture, complex state management, and seamless API integrations.

## Technologies Used

### Core
* **React (v18.3.1)** — UI library for building interactive interfaces via components and hooks.
* **Redux Toolkit (v2.2.5)** — Global state management for the Shopping Cart and Wishlist.
* **TypeScript (v5.2.2)** — Static typing for enhanced code safety and error catching.
* **SASS / SCSS (v1.77.6)** — Modular styling with variables, mixins, and nested selectors.

### UI/UX & Logic
* **React Router (v6.23.1)** — Declarative routing for multi-page navigation and dynamic URLs.
* **State Persistence** — `LocalStorage` integration to preserve user data across sessions.
* **Data Fetching** — Asynchronous API requests with loading skeletons and error handling.
* **Responsive Design** — Fluid CSS Grid and Flexbox layouts optimized for all devices.

### Development & Deployment
* **Vite (v5.2.11)** — High-performance frontend bundler and development server.
* **ESLint (v8.57.0)** — Code linting to enforce best practices and clean syntax.
* **GitHub Pages** — Automated project hosting and continuous static deployment.


## Getting Started
Follow these instructions to set up and run the project locally on your machine:

1. Clone the repository:
```bash
git clone https://github.com/nice_gadgets.git
cd nice_gadgets
```
2. Install the necessary dependencies:
```bash
npm install
```
3. Run the project locally in development mode:
```bash
npm start
```

This will start a local development server, and you can view the website in your browser.
